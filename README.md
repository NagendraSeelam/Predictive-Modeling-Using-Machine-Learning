# Predictive Modeling Using Machine Learning

## Objective
Build a machine learning model to predict outcomes from data.

## Dataset
Iris Dataset

## Algorithm Used
Decision Tree Classifier

## Process
1. Load dataset
2. Split training and testing data
3. Train model
4. Predict outcomes
5. Evaluate accuracy
6. Generate confusion matrix

## Libraries
- Pandas
- Scikit-learn
- Matplotlib

## Outcome
Successfully built and evaluated a machine learning classification model.
